## Things to install and get ready in Windows
* Java JDK
* Chrome Driver
* IntelliJ IDEA
* Selenium

# Setup Java JDK
**Step 1 – Download and Install JavaJDK from Oracle**
```
1. Download from https://www.oracle.com/technetwork/java/javase/downloads/jdk12-downloads-5295953.html
2. Install the downloaded file above
```

**Step 2 – Crosscheck in the System Environment Variables**
```
1. Open System Environment Variables in Windows Search Menu
2. Click Environment Variables
3. Click New in System Variables
4. Type JAVA_HOME as Variable Name
5. Copy your Java JDK path as Variable Value
6. Click OK
7. Find Path variable in System Variables
8. Click Edit
9. Please make sure you have %JAVA_HOME%\bin on the list
10. If you don't have %JAVA_HOME%\bin on the list, just add it
11. Click OK
```

# Setup Chrome Driver
**Step 1 – Locate the chromedriver**
```
1. Download from https://chromedriver.chromium.org/downloads based on your Google Chrome version
2. Locate the downloaded file to your local directory, then later call this directory while running the driver
```

# Setup IntelliJ IDEA
**Step 1 – Install IntelliJ IDEA**
```
1. Download from https://www.jetbrains.com/idea/download/download-thanks.html?platform=windows&code=IIC
2. Install the downloaded file above
```

# Setup Selenium
**Step 1 – Download Selenium Plugin**
```
1. Open Setting in your IntelliJ IDEA (CTRL+Alt+S)
2. Go to plugin
3. Download and install Selenium Plugin
4. Restart your IDE
```

**Step 2 – Download SeleniumRC Server**
```
1. Open Setting in your IntelliJ IDEA (CTRL+Alt+S)
2. Go to plugin
3. Download and install SeleniumRC Server
4. Restart your IDE
```
